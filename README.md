# 🛰️ Zcore-Framework: Ultimate Global PC Edition (v1.0)

Developed by **Umut** An advanced, multi-threaded open-source cybersecurity intelligence and endpoint auditing platform built entirely in Python. 

Zcore-Framework bridges the gap between lightweight mobile reconnaissance and powerful desktop infrastructure mapping. It consolidates **25 active modules** under a clean, unified command matrix.

---

## ⚡ Main Feature Matrix (25 Active Tools)

### 🌐 Section 1: External OSINT & Domain Recon
1. **My Public External IP:** Instantly fetch external IP address network metadata.
2. **Domain to IP Resolver:** Automate mapping of URLs to bare sunucu IPv4 strings.
3. **IP to Nameserver (Reverse DNS):** Interrogate active PTR infrastructure records.
4. **WHOIS Registrar Core Lookup:** Trace hosting entity registration databases.
5. **Geo-Location Radar Mapping:** Extract coordinates, city data, and international regional maps.

### 🛡️ Section 2: Advanced Port Auditing
6. **Stealth Port Scan:** Blazing fast checks on top critical network ports (21, 22, 80, 443, 445).
7. **System Services Scan:** Thoroughly audit all standardized privilege ports (1-1024).
8. **Gaming & Backdoor Audit:** Targets specific backdoors, unauthenticated panels, and game endpoints.
9. **Custom Target Port Scanner:** Isolate and target specific ports or personalized ranges.
10. **Banner Grabbing Core Tester:** Force communication channels to disclose application headers.

### 🛰️ Section 3: LAN & Infrastructure Radar
11. **Live Host Ping Sweep:** Discover all online assets across your local gateway in parallel threads.
12. **LAN Service Discovery Scanner:** Map out surrounding computers hosting accessible SMB/HTTP servers.
13. **Gateway Protocol Finder:** Extract routing tables to verify primary modem IPs.
14. **MAC Vendor System Identifier:** Decode physical hardware addresses to determine device brands.
15. **Subnet Mask Host Calculator:** Instantly compute network boundary thresholds.

### 🕵️ Section 4: ScamHunter Anti-Fraud
16. **Phishing URL Engine Cleaner:** Strip track codes, tracking strings, and nested HTTP subdomains.
17. **Cloudflare WAF Proxy Detector:** Flag fraudulent sites masking their authentic origin servers.
18. **Hosting Reputation Check:** Cross-reference networks to identify servers prone to housing fraud loops.
19. **SSL Certificate Validity Test:** Inspect transport layer encryption safety stamps.
20. **Bulk Link Threat Analyzer Loop:** Process massive queues of suspicious links via sequential checks.

### 💻 Section 5: Endpoint System Defense
21. **SMB Share Integrity Auditor:** Flag unauthenticated administrative network shares (`C$`, `ADMIN$`).
22. **Active Interface Adapter Lister:** Extract internal IP ranges dynamically allocated to hardware adapters.
23. **DNS Connection Trace Cache Extractor:** Mine transient system caches to view recent browser paths.
24. **Wipe Session Intelligence Data:** Instantly delete or reset the localized session report (`zcore_report.txt`).
25. **Secure Core Framework Logout:** Safely close operations and log out of the global control grid.
